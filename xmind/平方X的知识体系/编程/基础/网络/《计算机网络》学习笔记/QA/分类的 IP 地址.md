    32 位的 IP 地址
    A 类 0   8-1 位网络号    24 位主机号
    B 类 10  16-2 位网络号   16 位主机号  
    C 类 110 24-3 位网络号   8 位主机号
    
# C 类
    0b 1100 0000=192
    0b 1111 1111=255
    
# B 类
    0b 1000 0000=128
    0b 1011 1111=191
    
# A 类
    0b 0000 0000=0      全 0 表示 this
    0b 0111 1111=127    环回测试
