# 算法概述
[冒泡排序](https://zh.wikipedia.org/zh/%E5%86%92%E6%B3%A1%E6%8E%92%E5%BA%8F)

    i∈[0,n-1)
        j∈[0,n-1-i)
            swap(j,j+1)

# 时间复杂度
O(n^2)

外层 [0,n-1) 共 n 轮
内层 从 n-1 次到 1 次
共 (n-1+1)*n/2=n^2/2

## 平均情况
O(n^2)

## 最坏情况
O(n^2)

倒序时

## 最好情况
O(n)

正序时，记录未交换不再进行循环

# 空间复杂度
O(1)

i,j 以及可能的 swap,t

# 稳定性
稳定

相等时不会交换
