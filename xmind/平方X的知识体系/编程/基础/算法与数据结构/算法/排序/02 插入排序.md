    def sort(self, a: List) -> List:
        for i in range(1, len(a)):  # i 作为要查找插入的值
            t = a[i]  # 值
            j = i  # 开始比较的数
            while j > 0 and t < a[j - 1]:
                # 如果t小，则值向后移
                a[j] = a[j - 1]
                j -= 1
            # 当退出循环时，j 为 0 或者 t 比 a[j-1] 大
            a[j] = t
        return a
    
    
时间复杂度以交换作为基本语句，
如果比较操作的代价比交换操作大的话，可以采用二分查找法来减少比较操作的数目。该算法可以认为是插入排序的一个变种，称为二分查找插入排序。

# 时间复杂度
O(n^2)

## 最好情况
O(n)
每一轮比较第一个就退出
