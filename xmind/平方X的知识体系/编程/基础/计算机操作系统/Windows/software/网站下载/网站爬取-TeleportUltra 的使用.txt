# 下载部分文件
https://developer.android.google.cn/reference/android/graphics/ImageDecoder

软件会认为 ImageDecoder 是文件，于是下载 graphics 目录下的所有文件。

深度设为 0 会只下载单个文件

勾选后相关方件会有后缀，.htm 如果不勾，则如果原地址无后缀就无后缀。
