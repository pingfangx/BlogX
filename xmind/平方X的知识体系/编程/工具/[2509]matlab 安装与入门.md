# 基本语法
一开始不太理解的，在 getstart 里面有介绍  
data(:,1:end-1)';  
逗号分隔了 行,列  
: 表示范围  
end 是自带函数  
' 表示转置
  
别的应该没什么，有自带的 getstart.pdf 以及 shift+F1 查看函数浏览器，在右键中都很完整。
