# 0x01如何点击超链接
自己点击的时候，一直打不开，按Control也不行……  
查了一下  
[请问notepad++能直接点击超链接吗？](http://tieba.baidu.com/p/3732677764)  
在设置——首选项——其他——超链接设置——启用中启用（默认是启用的），然后双击就可以了。  
不过要注意，如果在markdown中，使用方括号加圆括号的形式，双击的链接会带上“)”，可能造成无法打开。
